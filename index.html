Selected project

Arduino Environmental Monitor
Real-time temperature & humidity logger with OLED display
Read write-up

An Arduino-based IoT project that reads temperature and humidity from a DHT22 sensor, displays values on a 128x64 OLED, and logs data to an SD card. Tech: Arduino (C++), DHT22, SSD1306 OLED, SD card module.

Arduino Greenhouse Controller
Automated watering and lighting system with sensors
GitHub

An Arduino-based greenhouse automation system that controls water pumps and LED grow lights based on soil moisture and light levels. Features an LCD display showing real-time sensor data and SD card logging. Tech: Arduino, moisture sensors, light sensors, relays.

Arduino Weather Station
Multi-sensor weather monitoring and data logging
Live Demo

A comprehensive weather station built with Arduino that measures temperature, humidity, air pressure, and UV index. Displays readings on a web interface and logs data to the cloud. Tech: Arduino, DHT22, BMP280, VEML6070, WiFi shield.

Arduino Gas Leak Detector
Real-time gas sensing with alarm and email alerts
Demo

A safety system using Arduino and MQ-5 gas sensor that detects natural gas leaks, triggers a piezo buzzer alarm, and sends email notifications. Includes LCD display for PPM readings. Tech: Arduino, MQ-5 sensor, Ethernet module.

Arduino Motion-Triggered Camera
PIR sensor captures images to SD card
Source

A motion detection camera system using Arduino, PIR motion sensors, and camera module that automatically captures images when motion is detected. Images are saved with timestamps to an SD card. Tech: Arduino, PIR sensor, OV7670 camera.

Arduino Line-Following Robot
Autonomous robot using IR sensors and DC motors
Repo

A mobile robot that autonomously follows black lines on white surfaces using IR sensors and speed control via PWM. Features obstacle avoidance and adjustable line detection threshold. Tech: Arduino, IR sensors, DC motors, motor driver.
